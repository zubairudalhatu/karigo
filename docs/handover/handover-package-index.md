## Start Here

1. Read `final-mvp-handover-summary.md`.
2. Read `final-provider-readiness-matrix.md` and `provider-go-live-decision-plan.md`.
3. Read `final-launch-risk-register.md` and `launch-blocker-audit.md`.
4. Follow `local-demo-runbook.md` and use `docs/demo/demo-script.md`.
5. Complete `internal-demo-approval-checklist.md`.
6. Provision and approve staging using `staging-environment-recommendation.md`.
7. Complete the private staging deployment decision and validation records.
8. Run `docs/qa/full-staging-pilot-simulation-plan.md` and private staging execution records.
9. Complete `controlled-soft-launch-approval-checklist.md` before authorising customer traffic.
