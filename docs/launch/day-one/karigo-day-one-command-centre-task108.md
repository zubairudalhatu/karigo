## Command Centre Rules

- One Pilot Lead makes final continue/pause/stop decisions.
- Every issue gets a severity and owner.
- P0/P1 issues pause the affected flow immediately.
- No sensitive evidence is pasted into command centre chat.
- APK links are shared only through the approved pilot channel.
- Live providers remain off unless a specific test-mode decision is recorded.
