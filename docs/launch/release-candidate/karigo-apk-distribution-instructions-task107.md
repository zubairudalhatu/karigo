# KariGO APK Distribution Instructions - Task 107

Date prepared: 2026-07-13

Do not publish APK links publicly during controlled early access. Share APK links only
through the approved private pilot distribution channel.

## Customer App APK

1. Download the approved Customer App APK only from the official Expo/EAS build link:
   [CUSTOMER_APK_BUILD_URL].
2. Do not forward unapproved APK builds.
3. Install on the pilot Android device.
4. Allow install from trusted source if Android requests permission.
5. Confirm the app opens.
6. Confirm the user can log in or sign up.
7. Confirm app version/build note if visible.
8. Report installation issues to operations support:
   [OPERATIONS_SUPPORT_PLACEHOLDER].

## KariGO Captain APK

1. Download the approved KariGO Captain APK only from the official Expo/EAS build link:
   [CAPTAIN_APK_BUILD_URL].
2. Uninstall any old KariGO Rider/Captain app first if required by operations support.
3. Install the fresh APK.
4. Confirm the app name shows KariGO Captain.
5. Confirm Delivery Captain mode appears.
6. Confirm Ride Captain remains readiness-only.
7. Report installation issues to operations support:
   [OPERATIONS_SUPPORT_PLACEHOLDER].

## Distribution Rules

- Share only release-candidate links approved in the release-candidate approval log.
- Do not upload APKs to public drives, public websites or social media.
- Do not commit APK/AAB files to Git.
- Do not send passwords, OTPs, delivery codes or credentials with APK links.
- If a wrong APK is shared, pause distribution and notify the pilot lead immediately.
