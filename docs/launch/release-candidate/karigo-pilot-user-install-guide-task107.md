# KariGO Pilot User Install Guide - Task 107

Date prepared: 2026-07-13

This guide is for controlled Kano pilot users only. Do not include passwords in this
document or any Git-tracked file.

## Customers

### Install The App

1. Open the official KariGO pilot app link shared by the KariGO team:
   [CUSTOMER_APP_LINK].
2. Download and install the Customer App APK.
3. If Android asks for permission, allow install from the trusted source used by the
   KariGO team.
4. Open the app and confirm the KariGO welcome/home screen appears.

### Sign Up Or Log In

1. Use the phone number approved for pilot access.
2. Sign up or log in through the app.
3. Do not share your password or OTP with anyone.

### Place First Order

1. Browse selected vendors.
2. Add item(s) to cart.
3. Confirm delivery address and server pricing.
4. Create order.
5. Complete mock payment or approved Paystack Test Mode only if instructed by KariGO.

### Track Order

1. Open Orders.
2. Select the active order.
3. Watch status updates as vendor, admin and Delivery Captain progress the order.

### Share Delivery Code Safely

Only share your delivery code after you have received your order. Do not send the code in
chat or screenshots.

### Report A Problem

Use in-app support or the approved pilot support channel:
[SUPPORT_CONTACT_PLACEHOLDER].

## Delivery Captains

### Install KariGO Captain

1. Open the official KariGO Captain APK link shared by the KariGO team:
   [CAPTAIN_APP_LINK].
2. Download and install the APK.
3. Open the app and confirm the app name shows KariGO Captain.

### Log In

1. Use your approved pilot login.
2. Do not share your password or OTP.

### Check Delivery Captain Mode

1. Confirm Delivery Captain mode is visible.
2. Confirm Ride Captain remains readiness-only.
3. Go online only when available.

### Accept Assigned Delivery

1. Open assigned delivery/job details.
2. Review pickup and delivery information.
3. Accept only jobs you can complete safely.

### Complete Pickup And Delivery

1. Follow the app status steps.
2. Confirm pickup from the vendor.
3. Progress the delivery status.
4. Ask for the customer delivery code only after the customer receives the order.
5. Complete delivery using the customer-supplied code.

### Report Issue

Report app, assignment or delivery issues to:
[OPERATIONS_SUPPORT_PLACEHOLDER].

## Vendors

### Use Vendor Dashboard

Open the Vendor Dashboard:
https://vendor.karigo.com.ng

### Login

Use the approved vendor login shared through the private pilot channel. Do not record or
share passwords in documents or chats.

### Check Products

1. Confirm product list.
2. Confirm availability and prices.
3. Report incorrect product details to operations.

### Accept Order

1. Open new paid orders.
2. Accept only available orders.
3. Reject only through the approved reason flow if needed.

### Prepare Order

1. Mark order preparing.
2. Prepare/package the order.

### Mark Ready For Pickup

Mark the order ready for pickup only when the package is ready for the assigned Delivery
Captain.

### Report Issue

Use the approved vendor support channel:
[VENDOR_SUPPORT_PLACEHOLDER].
