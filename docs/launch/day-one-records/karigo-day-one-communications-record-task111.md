# KariGO Day-One Communications Record - Task 111

## Purpose

Use this record to document Day-One pilot communications sent manually to customers,
vendors, Delivery Captains, admins, management and support stakeholders.

This record does not activate automated SMS, email, WhatsApp, push notifications or any
provider integration. Termii, WhatsApp, email and push remain disabled unless separately
approved in a future task.

## Communication Rules

- Keep messages short, accurate and operational.
- Do not share passwords, OTPs, access tokens, private addresses, payment secrets,
  webhook details, provider keys or unmasked private phone numbers.
- Do not promise live payments, live utilities, live rides, wallet withdrawals, automatic
  refunds, payouts or Pharmacy marketplace availability.
- If a message refers to payment, state that the Day-One pilot uses mock payment.
- If a message is sent to a participant group, record the group name rather than full
  recipient details.

## Day-One Message Templates

### Pilot Start Notice

Hello. KariGO's controlled Kano pilot window is now open for selected participants. Please
use only the approved app or dashboard access shared with you. Today's pilot uses mock
payment and is monitored by the KariGO operations team.

### Vendor Readiness Check

Hello. Please confirm your team is ready to receive KariGO pilot orders today. Keep your
dashboard open and update each order status promptly.

### Delivery Captain Readiness Check

Hello. Please confirm you are available for KariGO pilot delivery assignments today.
Only accept assignments you can complete safely and report any issue immediately.

### Temporary Pause Notice

KariGO has temporarily paused new pilot orders while the operations team reviews an
issue. Existing participants will receive an update before the pilot resumes.

### Resume Notice

KariGO pilot operations have resumed. Please continue using the approved pilot access
and report any issue to the support/operations contact.

### Day-One Close Notice

Thank you for participating in KariGO's Day-One controlled Kano pilot. The operations
team will review today's results before confirming the next pilot window.
