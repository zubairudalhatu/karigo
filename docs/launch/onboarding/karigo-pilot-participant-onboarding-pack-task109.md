# KariGO Pilot Participant Onboarding Pack - Task 109

Date prepared: 2026-07-13
Pilot location: Kano
Pilot duration: 7 to 14 days

## Purpose

Prepare real controlled early access participants for the Kano pilot. This pack covers
participant onboarding, access-control rules, role readiness and safe distribution
procedures before sending access to pilot users.

This is a launch operations and onboarding document only. It does not activate live
Paystack, live Accelerate.ng utilities, live Termii SMS, wallet withdrawals, automatic
refunds, live rides, ride dispatch, payouts, Pharmacy marketplace or provider login.

## Pilot Scope

| Group | Target count | Access surface |
| --- | --- | --- |
| Pilot vendors | 3 to 5 | Vendor Dashboard |
| Pilot customers | 20 to 50 | Customer App |
| Delivery Captains | 3 to 5 | KariGO Captain App |
| Operations admins | 2 to 3 | Admin Portal |
| Pilot duration | 7 to 14 days | Controlled early access |

## Onboarding Principles

- Invite only approved Kano pilot participants.
- Share access only through approved private channels.
- Do not publish APK links publicly.
- Do not record passwords, OTPs, delivery codes, bearer tokens or full private contact
  details in Git-tracked files.
- Confirm every participant understands feedback, support and privacy rules.
- Remove access quickly if a participant leaves the pilot or shares access improperly.

## Role-Specific Onboarding

### Customers

- Install the approved Customer App APK or current update.
- Sign up or log in with approved pilot details.
- Place orders only during agreed pilot windows.
- Share delivery code only after receiving the order.
- Report issues through the approved support channel.

### Vendors

- Log in to Vendor Dashboard at `https://vendor.karigo.com.ng`.
- Confirm products, availability and prices.
- Keep dashboard open during pilot windows.
- Accept only available orders.
- Mark preparing and ready for pickup accurately.
- Automated payout is not live during controlled early access.

### Delivery Captains

- Install approved KariGO Captain APK.
- Confirm app name shows KariGO Captain.
- Confirm Delivery Captain mode is visible.
- Go online only when available.
- Ask for delivery code only after the customer receives the order.
- Ride Captain is readiness-only; live ride trips are not active.

### Operations Admins

- Log in to Admin Portal at `https://admin.karigo.com.ng`.
- Monitor orders, dispatch, support, wallets, referrals and SME Services as assigned.
- Do not export or share sensitive user data outside approved pilot process.
- Escalate P0/P1 issues immediately.

## Participant Exit

Remove or disable participant access when:

- pilot participation ends;
- participant requests removal;
- access is shared outside the approved group;
- suspicious activity or privacy concern appears;
- management pauses or stops the pilot.

See `karigo-access-revocation-and-incident-procedure-task109.md`.
