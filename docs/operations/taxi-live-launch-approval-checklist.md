# Taxi Live Launch Approval Checklist

Taxi dispatch is staging-only until this checklist is approved by management, operations, legal and technical owners.

## Product Approval

- [ ] Taxi booking scope approved.
- [ ] Launch city and zones approved.
- [ ] Customer booking flow reviewed.
- [ ] Driver workflow reviewed.
- [ ] Admin dispatch workflow reviewed.
- [ ] Trip cancellation policy approved.
- [ ] Fare display and receipt wording approved.

## Legal and Compliance

- [ ] Local Taxi/transport regulation reviewed.
- [ ] Driver eligibility requirements approved.
- [ ] Vehicle inspection requirements approved.
- [ ] Insurance requirements approved.
- [ ] Customer Taxi terms approved.
- [ ] Driver Taxi agreement approved.
- [ ] Data protection review completed.
- [ ] Emergency/SOS process reviewed.

## Driver Operations

- [ ] Driver identity checks completed.
- [ ] Driver licence checks completed.
- [ ] Vehicle ownership or authorization checked.
- [ ] Vehicle inspection completed.
- [ ] Driver training completed.
- [ ] Driver code of conduct accepted.
- [ ] Incident reporting process explained.
- [ ] Approved driver list signed off.

## Fare and Payment

- [ ] Fare policy approved.
- [ ] Maps/distance source approved.
- [ ] Surge or no-surge decision approved.
- [ ] Payment provider live approval completed.
- [ ] Refund/cancellation rules approved.
- [ ] Driver payout process approved.
- [ ] Reconciliation process approved.

## Maps and Safety

- [ ] Maps/geolocation provider selected.
- [ ] Provider billing approved.
- [ ] API keys stored in secret manager only.
- [ ] Location privacy policy reviewed.
- [ ] Emergency contact process ready.
- [ ] Safety escalation matrix ready.
- [ ] Incident response template ready.

## Technical Go-Live

- [ ] Staging Taxi tests completed.
- [ ] Trip lifecycle tests passed.
- [ ] Trip PIN tests passed.
- [ ] Admin assignment tests passed.
- [ ] Driver availability tests passed.
- [ ] Feature flags reviewed.
- [ ] Production flags remain off until final approval.
- [ ] Monitoring/logging ready.
- [ ] Rollback plan ready.

## Decision

- [ ] Approved for extended staging test.
- [ ] Approved for controlled Taxi pilot.
- [ ] Approved for public Taxi launch.
- [ ] Not approved due to blockers.
